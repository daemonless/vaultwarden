## Deployment

### Podman Compose

```yaml
services:
  vaultwarden:
    image: ghcr.io/daemonless/vaultwarden:latest
    container_name: vaultwarden
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=UTC
      - SIGNUPS_ALLOWED=true
    volumes:
      - "/path/to/containers/vaultwarden:/config"
    ports:
      - 80:80
    restart: unless-stopped
```
